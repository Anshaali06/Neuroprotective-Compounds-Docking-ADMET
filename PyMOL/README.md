# PyMOL Visualization

This folder contains the protein–ligand visualization generated using PyMOL for the docked Acetylcholinesterase (PDB ID: 4EY7)–Quercetin complex.

## Files

### Quercetin_Docking_Final.png

Overall view of the docked protein–ligand complex.

* Protein: Cyan cartoon representation
* Quercetin: Yellow stick representation
* Binding Pocket Residues: Green stick representation
* Resolution: 2500 × 1800 pixels (300 DPI)

### Quercetin_Binding_Site.png

Zoomed view of Quercetin occupying the active-site binding pocket of Acetylcholinesterase.

This image highlights the interaction region between the ligand and surrounding amino acid residues involved in binding.

## Summary

PyMOL visualization confirmed that Quercetin was successfully accommodated within the active-site cavity of Acetylcholinesterase (4EY7), supporting the molecular docking results obtained from AutoDock Vina.
